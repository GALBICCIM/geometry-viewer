# Geometry Viewer

A project for visualizing 3D models.

⚠️ Do not rename the sample file ⚠️

## How to Use

```bash
git clone https://github.com/GALBICCIM/geometry-viewer.git
cd geometry-viewer
npm i
npm run dev
```

## Supported File Formats

- `.vtp`
- `.h5`

## Tech Stack

- React + Typescript
- Babylon.js
- VTK.js
- Zustand
- styled-components
